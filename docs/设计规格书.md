# OTU Tech Product Catalog — Design Specification

> 2026-06-14 | v2.0

## Overview

A dark-themed product showcase web app for tech consumer electronics. Pure display, no transactions. Users browse products and copy a WeChat ID to contact the seller.

**Primary deliverable**: `index.html` — a single-file, zero-dependency web application.
**Reference**: `miniprogram/` — original WeChat Mini Program prototype (not functional without WeChat Cloud).

## Design Direction: Dark Tech

Deep black background with cyan accents. Tech products (earbuds, keyboards, mice) look best on dark surfaces — clean silhouettes, focused visual hierarchy.

| Token | Value | Usage |
|-------|-------|-------|
| `--bg-primary` | `#0a0a0f` | Page background |
| `--bg-card` | `#14141f` | Card surfaces |
| `--accent` | `#00e5ff` | Prices, buttons, interactive states |
| `--accent-warm` | `#ff6d00` | Tags, badges |
| `--text-primary` | `#f5f5f5` | Headings, body text |
| `--text-secondary` | `#9e9e9e` | Labels, metadata |
| `--divider` | `#1e1e2e` | Borders, separators |

Each category has a unique gradient background and emoji icon for visual differentiation without images.

### Category Page

- Horizontal scrolling category tabs (12 categories)
- Product count per category
- 2-column product card grid
- Active tab highlighted with accent color

### Search Page

- Auto-focusing search input with clear button
- Hot keyword tags (shown before input)
- Real-time search: matches `name` and `description` fields
- Results as single-column list (icon + name + brand + price)

### Product Detail

- Large product image area (emoji placeholder with category gradient)
- Product name, price, brand/model/category metadata
- Description section (if available)
- Specs table (key-value pairs)
- Fixed bottom bar: "Copy WeChat ID" button

### About Page

- Brand name and description
- Total product count (dynamic from data)
- WeChat ID contact card (tap to copy)

## Data Model

### `products.js`

Static data file loaded via `<script>` tag. No runtime dependencies.

```js
{
  id: "AU-01",              // unique identifier
  name: "65W GaN 充电器",   // product name
  price: 129,               // price in CNY (0 = "询价" / inquiry)
  brand: "声阔",            // brand
  model: "G65",             // model number
  category: "充电器",       // one of 12 categories
  description: "65W...",    // product description
  specs: [                  // specifications
    { k: "功率", v: "65W" },
    { k: "接口", v: "USB-C × 2" }
  ],
  isHot: true,              // featured on home page
  sort: 100                 // display weight (higher = first)
}
```

### `categories`

Array of 12 category strings, used for tab rendering and filtering.

## Architecture

```
index.html
├── <style>         — all CSS (custom properties, components, layout)
├── <body>          — HTML structure (phone frame, pages, tabs)
└── <script>
    ├── Data        — loaded from products.js
    ├── State       — currentPage, activeCategory, pageStack
    ├── Render      — product cards, category tabs, search results
    ├── Navigation  — showPage, goBack, switchTab, showDetail
    ├── Search      — doSearch, quickSearch, clearSearch
    └── Actions     — copyWechat, showToast, banner auto-rotation
```

### Navigation Model

- **Tab switching**: resets pageStack, shows target tab
- **Page navigation**: pushes currentPage to pageStack, shows new page
- **Back button**: pops pageStack, shows previous page
- Back button visibility: shown when pageStack is non-empty

### Page Transitions

CSS `@keyframes fadeIn` animation (0.2s ease) on page switch for smooth visual feedback.
