# 微信小程序产品展示 — 设计规格书

> 2026-06-14 | 版本 v1.0

## 项目概述

科技消费品展示目录小程序。纯展示、无交易，用户浏览产品后一键复制微信号联系。
微信原生开发 + 云开发数据库。

## 核心要素

| 项 | 决定 |
|----|------|
| 品类 | 耳机 / 鼠标 / 键盘 / 配件 |
| 规模 | 50+ 款产品 |
| 用途 | 纯展示目录，无交易 |
| 联系 | 一键复制微信号 |
| 技术 | 微信原生 WXML + WXSS + JS，微信云开发（数据库 + 云函数） |
| 费用 | 云开发免费额度内（2GB 数据库 + 5GB 存储） |

## 一、设计方向：Dark Tech

**为什么：** 科技产品（耳机/鼠标/键盘）在深色背景上表现力最强，产品轮廓清晰，视觉焦点集中。与大多数微信小程序亮色路线拉开差距。

- **底色** `#0a0a0f` — 深黑，产品图片是唯一视觉焦点
- **强调色** `#00e5ff` 荧光青 — 价格、按钮、交互状态
- **次级强调** `#ff6d00` 暗橙 — 标签、徽章
- **文字** `#f5f5f5`（主）/ `#9e9e9e`（辅） — 高对比但不刺眼
- **字体** 系统默认（微信小程序限制），层级靠大小和粗细区分
- **间距** 大留白，卡片不设边框和阴影，靠间距和图片撑场

## 三、数据模型

### 云数据库集合：`products`

```js
{
  _id: "auto",           // 云数据库自动生成
  name: "Pro X1 降噪耳机",  // 产品名称
  price: 299,            // 价格（数字）
  brand: "声阔",         // 品牌
  model: "X1",           // 型号
  category: "耳机",      // 分类（耳机/鼠标/键盘/配件）
  description: "40dB主动降噪...",  // 产品描述
  specs: [               // 规格列表
    { key: "驱动单元", value: "40mm" },
    { key: "频率响应", value: "20Hz-20kHz" },
    { key: "连接方式", value: "蓝牙5.3 / 有线" }
  ],
  images: [               // 产品图片（云存储 fileID 数组）
    "cloud://xxx.png",
    "cloud://xxx.png"
  ],
  isHot: false,          // 是否为热门推荐
  sort: 0,               // 排序权重，越大越前
  createdAt: Date         // 创建时间
}
```

### 云函数

| 函数 | 用途 |
|------|------|
| `getProducts` | 按分类/热门/分页查产品列表 |
| `searchProducts` | 按关键词搜索产品 |

### 页面数据流

```
页面 onLoad
  → wx.cloud.callFunction(getProducts)
    → 云函数查 products 集合
      → 返回 JSON
        → setData 渲染
```

## 四、技术要点

1. **图片管理**：产品图片上传到微信云存储，数据库只存 `fileID`。页面用 `fileID` 换取临时链接渲染。
2. **性能**：列表页分页加载（每页 10 条），图片懒加载 `lazy-load`。
3. **搜索**：前端调云函数，用正则关键词匹配 `name` 和 `description`。
4. **微信号配置**：微信号存云数据库独立集合 `config`，一处修改全局生效。
5. **审核注意**：纯展示类小程序类目选「工具 > 信息查询」，无需特殊资质。
